EEG - MOTOR IMAGERY Classification

This project classifies Motor Imagery (MI) tasks from EEG signals using MATLAB as part of my MSc thesis. It demonstrates signal processing, feature extraction, and machine learning for brain-computer interfaces (BCI) .

Toolboxes & Libraries: Signal Processing Toolbox, Statistics and Machine Learning Toolbox, EEGLAB (for EEG handling)

Processing Steps:

  • Preprocessing: Filter and segment EEG signals for MI tasks.
  • Feature Extraction: Compute frequency band powers, CSP (Common Spatial Patterns), and connectivity measures.
  • Classification: Train models (LDA, SVM, Decision Tree) on extracted features.
  • Evaluation: Assess accuracy and performance across tasks.

Outcome: Achieved robust classification of MI tasks (~80%), demonstrating effective feature extraction, preprocessing pipelines, and coding skills for EEG analysis.

EMG Analysis – MEP Latency & Amplitude

This project analyzes motor-evoked potentials (MEPs) using MATLAB as part of my MSc thesis. Preprocessing, handling missing pulses, and extracting MEP latencies & amplitudes were performed using custom scripts.

Toolboxes & Libraries: MATLAB, Signal Processing Toolbox and Statistics

Analysis Steps: Preprocessing EMG data, detecting & replacing missing pulses, extracting latencies & amplitudes for Rest/Task trials, validating MEPs, and performing statistical analysis.

Outcome: Accurate MEP latency & amplitude extraction (~80% reliability), showcasing MATLAB coding, EMG preprocessing, and statistical analysis skills.

EEG Analysis – Epileptic Seizure Prediction

This project analyzes multi-channel EEG to predict seizures by examining effective connectivity (EC) changes. Using MATLAB, I developed scripts to preprocess data, extract frequency band features, compute Directed Transfer Function (DTF) connectivity, and evaluate metrics like sensitivity and false prediction rate. Custom plotting scripts visualize ictal and interictal signals.

Skills demonstrated: MATLAB coding, signal processing, connectivity analysis, automation of results, and data visualization. The DTF-based approach achieved high sensitivity (~80%) and low false prediction rate (0.33 FP/h), especially in gamma and beta bands.
